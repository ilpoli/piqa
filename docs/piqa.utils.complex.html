piqa.utils.complex.angle(x)#

Returns the angle (phase) of \(x\).

\[\phi(x) = \operatorname{atan2}(\Im(x), \Re(x))\]
Parameters

x (Tensor) – A complex tensor, \((*, 2)\).

Returns

The angle tensor, \((*,)\).

Return type

Tensor

Example

>>> x = torch.tensor([[2., 0.], [0.7071, 0.7071]])
>>> angle(x)
tensor([0.0000, 0.7854])
piqa.utils.complex.complx(real, imag)#

Returns a complex tensor with its real part equal to \(\Re\) and its imaginary part equal to \(\Im\).

\[c = \Re + i \Im\]
Parameters

  • real (Tensor) – A tensor \(\Re\), \((*,)\).

  • imag (Tensor) – A tensor \(\Im\), \((*,)\).

Returns

The complex tensor, \((*, 2)\).

Return type

Tensor

Example

>>> x = torch.tensor([2., 0.7071])
>>> y = torch.tensor([0., 0.7071])
>>> complx(x, y)
tensor([[2.0000, 0.0000],
        [0.7071, 0.7071]])

piqa.utils.complex.dot(x, y)#

Returns the element-wise dot-product of \(x\) and \(y\).

\[x \odot y = \Re(x) \Re(y) + \Im(x) \Im(y)\]
Parameters

  • x (Tensor) – A complex tensor, \((*, 2)\).

  • y (Tensor) – A complex tensor, \((*, 2)\).

Returns

The dot-product tensor, \((*,)\).

Return type

Tensor

Example

>>> x = torch.tensor([[2.,  0.], [0.7071,  0.7071]])
>>> y = torch.tensor([[2., -0.], [0.7071, -0.7071]])
>>> dot(x, y)
tensor([4., 0.])

  • squared (bool) – Whether the output is squared or not.

Returns

The modulus tensor, \((*,)\).

piqa.utils.complex.pow(x, exponent)#

\[x^p = \left| x \right|^p \exp(i \phi(x))^p\]
Parameters

  • x (Tensor) – A complex tensor, \((*, 2)\).

  • exponent (float) – The exponent \(p\).

Returns

The power tensor, \((*, 2)\).

Return type

Tensor

Example

>>> x = torch.tensor([[2., 0.], [0.7071, 0.7071]])

        [-4.3711e-08,  9.9998e-01]])
piqa.utils.complex.prod(x, y)#

Returns the element-wise product of \(x\) and \(y\).

\[x y = \Re(x) \Re(y) - \Im(x) \Im(y) + i \left( \Re(x) \Im(y) - \Im(x) \Re(y) \right)\]
Parameters

  • x (Tensor) – A complex tensor, \((*, 2)\).

  • y (Tensor) – A complex tensor, \((*, 2)\).

Returns

The product tensor, \((*, 2)\).

Return type

Tensor

Example

>>> x = torch.tensor([[2.,  0.], [0.7071,  0.7071]])
>>> y = torch.tensor([[2., -0.], [0.7071, -0.7071]])
>>> prod(x, y)
tensor([[4.0000, 0.0000],
        [1.0000, 0.0000]])
